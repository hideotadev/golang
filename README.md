### Go on macOS
```sh
curl -OL https://golang.org/dl/go1.16.4.darwin-amd64.tar.gz
tar xfz go1.16.4.darwin-amd64.tar.gz
sudo mv go /usr/local
sudo ln /usr/local/go/bin/go /usr/local/bin/go
```
